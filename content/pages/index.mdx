# UNiD Documentation

- UNiDが目指す世界観や概要について説明する


UNiDは、新たなDigital Trustを構築するための手段を提供します。これまでのインターネット上の信頼は、国家や認証局や巨大なIT企業などの中央集権組織を信頼することで成立していました。これらの信頼をグローバルに拡大するためには、プロトコルレベルでお互いを信頼し、安全にデータを交換できる仕組みが必要になります。Decentralized Identifiers (DIDs)は、グローバルな標準規格としてW3C, IETE, DIF and Hyperledger project at the Linux Foundationを中心に策定が進んでいます。UNiDはこのDIDsテクノロジーを応用したアプリケーション構築や、信頼のネットワークに参加するためのSDKを提供します。
