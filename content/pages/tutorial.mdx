# Tutorial

On this page, we get you up and running with UNiD SDK for embedding DID Core capabilities into your mobile or web applications. UNiD SDK allows you to reduce "honeypot" risks and improves user experience in cryptographically secure, privacy respecting, and machine-verifiable way.

If you don't already have an account and UNiD project established, head over to [unid.io](https://collabogate.com/products/index.html), then retune to this page.
