<SEO title='Credential Data Schema' />

The types related to 'Indetification' encapsulate any identification linking a person.

`IdentificationCredential` type.

**Properties tree**

```
.
|-- Identification
    |-- address
    |-- nationalID
        |-- nationality
        |-- identifier
```

### Identification

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| '@type' | `['Person', 'IdentificationPerson']` | true ||
| name | `string` | ture ||
| birthday | `date` | true ||
| gender | `string` | true ||
| email | `string` | true ||
| telephone | `string` | true ||
| image | `string` | true ||
| address | `array` | true ||
| nationality | `array` | true ||

### address
Extension of Identification with extensions.

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |

| addressLocality | `string` | true ||
| addressRegion | `string` | true ||

| streetAddress | `string` | true ||

### nationalID
Extension of Identification with extensions.

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| '@type' | `['Role', 'NationalityRole']` | true ||
| nationality | `array` | ture ||
| identifier | `array` | true ||

### nationality

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| '@type' | `Country` | true ||
| name | `string` | ture ||

### identifier

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| '@type' | `PropertyValue` | true ||
| propertyID | `string` | true ||
| value | `string` | true ||
