The types related to `Resume` encapsulate any resume linking a person.

```
.
|-- Resume
    |-- address
    |-- contactPoint
    |-- hasCredential
        |-- about
        |-- recognizedBy
    |-- worksFor
        |-- employee
            |-- hasOccupation
    |-- alumniOf
        |-- employee
            |-- hasOccupation
```

### ResumeCredentialV1

`ResumeCredentialV1` has the following properties and types.

| Property | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `ResumePerson` | true ||
| name | `string` | ture ||
| birthday | `date` | true ||
| email | `string` | true ||
| telephone | `string` | true ||
| image | `string` | true ||
| address | `PostalAddress` | true ||
| contactPoint | `ContactPoint` | true ||
| hasCredential | `EducationalOccupationalCredential` | true ||
| worksFor | `Organization` | true ||
| alumniOf | `Organization` | true ||

### alumniOf
Organizations that the person is an alumni of.

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `Organization` | true ||
| name | `string` | ture ||
| employee | `array` | true ||

### employee

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `Person` | true ||

### hasOccupation

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `EmployeeRole` | true ||
| roleName | `string` | ture ||
| startDate | `date` | true ||
| endDate | `date` | false ||
