The types related to `Resume` encapsulate any resume linking a person.

`ResumeCredential` type.

properties tree
```
.
|-- Resume
    |-- address
    |-- contactPoint
    |-- hasCredential
        |-- about
        |-- recognizedBy
    |-- worksFor
        |-- employee
            |-- hasOccupation
    |-- alumniOf
        |-- employee
            |-- hasOccupation
```

### Resume
Resume credential type.

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `["Person", "ResumePerson"]` | true ||
| name | `string` | ture ||
| birthday | `date` | true ||
| email | `string` | true ||
| telephone | `string` | true ||
| image | `string` | true ||
| address | `Address` | true ||
| contactPoint | `array<contactPoint>` | true ||
