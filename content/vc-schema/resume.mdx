<SEO title='Credential Data Schema' />

The types related to `Resume` encapsulate any resume linking a person.

`ResumeCredential` type.

properties tree
```
.
|-- ResumePerson
    |-- address
    |-- contactPoint
    |-- hasCredential
        |-- about
        |-- recognizedBy
    |-- worksFor
        |-- employee
            |-- hasOccupation
    |-- alumniOf
        |-- employee
            |-- hasOccupation
```

### ResumePerson

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
