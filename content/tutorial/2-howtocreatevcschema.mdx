Intro to create type extensions of credential subject and VCs.

## Service Account

Generally, a user account for a web service is created with the following attributions.

- name
- email
- gender
- birthDate
- telephone
- address

### Create a type extension of credential subject

```js
import * as Unid from "@unid/vc-unid-sdk";
type ServiceAccountCredentialV1 = Unid.createCredentialType(
    "@type": ["ServiceAccountPerson", "ServiceAccountOrganization"],
    "name": Unid.getCredentialType(NameCredentialV1),
    "birthDate": Unid.getCredentialType(BirthDateCredentialV1),
    "gender": Unid.getCredentialType(GenderCredentialV1),
    "image": Unid.getCredentialType(ImageCredentialV1),
    "email": Unid.getCredentialType(EmailCredentialV1),
    "telephone": Unid.getCredentialType(PhoneCredentialV1),
    "address": Unid.getCredentialType(AddressCredentialV1)
);
```

The type of credential can be referred in the same tenant network.

error message

### ServiceAccountCredentialV1 Schema

`ServiceAccountCredentialV1` is a type extension of credential subject into VC.

### ServiceAccountPerson
A service account of a person.

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `ServiceAccountPerson` | true ||
| name | `string` | ture ||
| birthDate | `date` | true ||
| gender | `string` | true ||
| image | `string` | true ||
| email | `string` | true ||
| telephone | `string` | true ||
| address | `PostalAddress` | true ||


### serviceAccount.jsonld file

```js
{
    "@context": {
        "@version": 1.1,
        "@protected": true,

        "id": "@id",
        "type": "@type",

        "ServiceAccountCredentialV1": {
            "@id": "https://docs.unid.plus/docs/2020/credentials/serviceAccount#ServiceAccountCredentialV1",
            "@context": {
                "@version": 1.1,
                "@protected": true,

                "id": "@id",
                "type": "@type",

                "ServiceAccountPerson": "http://schema.org/Person",
                "name": "http://schema.org/name",
                "birthDate": "http://schema.org/birthDate",
                "gender": "http://schema.org/gender",
                "email": "http://schema.org/email",
                "telephone": "http://schema.org/telephone",
                "image": "http://schema.org/image",
                "address": {
                    "@id": "http://schema.org/address",
                    "@type": "@id",
                    "@context": {
                        "@version": 1.1,
                        "@protected": true,

                        "id": "@id",
                        "type": "@type",
                        "PostalAddress": "http://schema.org/PostalAddress",
                        "addressCountry": "http://schema.org/addressCountry",
                        "addressLocality": "http://schema.org/addressLocality",
                        "addressRegion": "http://schema.org/addressRegion",
                        "postalCode": "http://schema.org/postalCode",
                        "streetAddress": "http://schema.org/streetAddress"
                    }
                }
            }
        }
    }
}
```








## ResumeCredentialV1 Schema

`ResumeCredentialV1` is a type extension of credential subject into VC.

### ResumePerson

| Property | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `ResumePerson` | true ||
| name | `string` | ture ||
| birthday | `date` | true ||
| email | `string` | true ||
| telephone | `string` | true ||
| image | `string` | true ||
| address | `PostalAddress` | true ||
| contactPoint | `ContactPoint` | true ||
| hasCredential | `EducationalOccupationalCredential` | true ||
| worksFor | `Organization` | true ||
| alumniOf | `Organization` | true ||

### alumniOf
Organizations that the person is an alumni of.

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `Organization` | true ||
| name | `string` | ture ||
| employee | `array` | true ||

### employee

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `Person` | true ||
| hasOccupation | `array` | ture ||

### hasOccupation

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `EmployeeRole` | true ||
| roleName | `string` | ture ||
| startDate | `date` | true ||
| endDate | `date` | false ||

## WorkTimeRecordCredentialV1

`WorkTimeRecordCredentialV1` is a type extension of credential subject into VC.

### WorkTimeRecordPerson

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `WorkTimeRecordPerson` | true ||
| workTimeRecord | `["JobPosting", "Role"]` | true ||


### ["JobPosting", "Role"]

| Attribute | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `["JobPosting", "Role"]` | true ||
| employmentType | `string` | ture ||
| startDate | `string` | true ||
| endDate | `string` | true ||
| Duration | `datetime` | false ||

    type: ["VerifiableCredential", "WorkTimeRecordCredentialV1"],
